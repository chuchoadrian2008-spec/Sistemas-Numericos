Sistema Decimal

¿Qué es el sistema decimal?

El sistema decimal es un sistema numérico de base 10, lo que significa que utiliza diez símbolos diferentes: 0, 1, 2, 3, 4, 5, 6, 7, 8 y 9.

Es el sistema más utilizado en la vida cotidiana para representar cantidades y realizar operaciones matemáticas.

Conversión de Decimal a otros sistemas

Decimal a Binario

El sistema binario utiliza solo los números 0 y 1.

Para convertir un número decimal a binario se siguen estos pasos:

  1. Dividir el número entre 2.
  2. Anotar el residuo.
  3. Volver a dividir el resultado entre 2.
  4. Repetir hasta que el cociente sea 0.
  5. Leer los residuos de abajo hacia arriba.

Ejemplo: Convertir 13 a binario

13 ÷ 2 = 6 residuo 1
6 ÷ 2 = 3 residuo 0
3 ÷ 2 = 1 residuo 1
1 ÷ 2 = 0 residuo 1

Resultado: 1101

Decimal a Octal

El sistema octal utiliza números del 0 al 7.

Pasos:

  1. Dividir el número entre 8.
  2. Anotar el residuo.
  3. Seguir dividiendo el cociente entre 8.
  4. Leer residuos de abajo hacia arriba.

Ejemplo: Convertir 25 a octal

25 ÷ 8 = 3 residuo 1
3 ÷ 8 = 0 residuo 3

Resultado: 31

Decimal a Hexadecimal

El sistema hexadecimal usa números del 0 al 9 y letras A, B, C, D, E y F.

  • A = 10
  • B = 11
  • C = 12
  • D = 13
  • E = 14
  • F = 15

Pasos:

  1. Dividir el número entre 16.
  2. Anotar el residuo.
  3. Si es mayor a 9, convertir a letra.
  4. Leer de abajo hacia arriba.

Ejemplo: Convertir 30 a hexadecimal

30 ÷ 16 = 1 residuo 14
1 ÷ 16 = 0 residuo 1

14 = E

Resultado: 1E
